# 讨论与分析规则库 (Discussion Rules)

## 讨论的层次结构

### Layer 1: 结果分析
- 结果是否合理？（与直觉/常识对比）
- 结果的关键特征是什么？（最大值、最小值、趋势、异常点）
- 结果的物理/实际意义是什么？

**模式**：可视化验证 → 发现差异 → 选择更优模型

### Layer 2: 模型对比
- 不同模型的优劣是什么？
- 在什么条件下模型A优于模型B？
- 选择最终模型的理由是什么？

**模式**：建立两种模型 → 分别求解 → 可视化对比 → 选择更优

### Layer 3: 一般性策略提炼
- 从具体结果中抽象出一般性规律
- 给出可迁移的策略/建议
- 讨论策略的适用条件

**模式**：具体求解 → 观察规律 → 抽象为一般策略 → 验证策略

### Layer 4: 局限性与展望
- 模型的假设条件有哪些？
- 假设不成立时会怎样？
- 如何改进？

## 讨论写作模板

### 结果分析模板
```
由[图/表X]可以看出，[描述关键特征]。
这表明[解释物理/实际意义]。
与[预期/常识/文献]对比，[分析差异原因]。
```

### 模型对比模板
```
本文建立了[模型A]和[模型B]两种模型。
[模型A]的特点是[...]，适用于[...]；
[模型B]的特点是[...]，适用于[...]。
通过[验证方法]对比，[模型X]更优，因为[理由]。
```

### 一般性策略模板
```
综合[所有结果/分析]，可以得到以下一般性策略/结论：
1. [策略1]：在[条件]下，应采取[行动]；
2. [策略2]：当[条件]时，应[行动]；
3. [策略3]：[行动]。
这些策略在[验证场景]中得到了验证。
```

---

## 禁止事项

1. **禁止只列结果不分析**：每个结果后必须有解释
2. **禁止主观臆断**：分析必须基于数据/公式
3. **禁止忽略异常值**：异常结果必须讨论原因
4. **禁止过度泛化**：结论不能超出模型适用范围
