# 模型选择规则库 (Model Selection Rules)

## 核心决策树

```
问题类型判断
├── 时间序列/预测类
│   ├── 趋势明显 → ARIMA / 指数平滑
│   ├── 非线性+多变量 → LSTM / GRU
│   └── 小样本 → 灰色预测 GM(1,1)
├── 评价/排序类
│   ├── 主观权重 → AHP / 模糊综合评价
│   ├── 客观权重 → 熵权法 / TOPSIS
│   └── 组合 → TOPSIS+AHP / 熵权-TOPSIS
├── 优化类
│   ├── 线性约束 → 线性规划(LP)
│   ├── 非线性+连续 → 非线性规划(NLP)
│   ├── 离散+组合 → 整数规划(IP) / MILP
│   ├── 多目标 → 主要目标法 / 加权法 / 帕累托
│   └── 大规模+启发式 → 遗传算法(GA) / 模拟退火(SA) / 粒子群(PSO)
├── 路径/规划类
│   ├── 最短路径 → Dijkstra / Floyd
│   ├── 覆盖问题 → 贪心算法 / 集合覆盖
│   └── 调度问题 → 动态规划 / 分支定界
├── 决策/博弈类
│   ├── 确定型多阶段 → 动态规划(DP)
│   ├── 随机型 → 马尔可夫决策过程(MDP)
│   ├── 多人竞争 → 博弈论(纳什均衡)
│   └── 序贯决策 → 树搜索 / 回溯
├── 分类/聚类类
│   ├── 有标签 → SVM / 随机森林 / XGBoost
│   └── 无标签 → K-means / DBSCAN / 层次聚类
└── 仿真类
    ├── 随机过程 → 蒙特卡洛仿真
    ├── 系统动力学 → SD建模
    └── Agent行为 → Agent-Based Model(ABM)
```

## 模型组合模式

### 递进式组合（由简到繁）
- 动态规划 → MDP → 博弈论 (B175)
- 二维几何 → 三维空间 → 多目标优化 (B311)

### 互补式组合（各取所长）
- 几何分析(建模) + 数值计算(求解) + CAD(验证) (B311)
- 贪心(局部最优) + 智能优化(全局最优) (B477)

### 预处理+建模组合
- 随机森林(数据预处理) + 几何建模(核心) + 智能算法(求解) (B477)

---

## 模型选择的论证要求

1. **必须说明选择理由**：为什么选这个模型而不是其他
2. **必须有对比或验证**：至少说明模型的适用性
3. **必须有局限性讨论**：模型的假设和适用范围

---

## 禁止事项

- 禁止无理由地选择模型
- 禁止堆砌模型（用多个模型但不说明关系）
- 禁止模型与问题不匹配（如用分类模型解决优化问题）
- 禁止忽略模型假设条件
