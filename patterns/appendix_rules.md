# 附录规则库 (Appendix Rules)

## 附录内容规范

### 必须包含
1. **核心代码**：求解模型的关键代码（Matlab/Python）
2. **详细数据**：正文未展示的完整数据表
3. **补充图表**：辅助理解的额外图表

### 不应包含
1. 大段文字说明
2. 与核心模型无关的代码
3. 原始数据（除非特别重要）

---

## 代码规范

### 格式要求
- 有注释说明关键步骤
- 变量名与正文符号一致
- 有输入输出说明

### 示例结构
```matlab
%% 问题1: 覆盖宽度计算
% 输入: D0-中心水深, alpha-坡度, theta-开角, x-距中心距离
% 输出: W-覆盖宽度, eta-重叠率

function [W, eta] = calc_coverage(D0, alpha, theta, x)
    D = D0 - x * tan(alpha);  % 式(4): 海水深度
    W = D * sin(theta/2) * (1/cos(theta/2+alpha) + 1/cos(theta/2-alpha));
    % ...
end
```
