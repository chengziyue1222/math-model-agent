# 摘要模式库 (Abstract Patterns)

## 核心结构

摘要必须包含四要素：**问题→方法→结果→价值**

---

## Pattern 1: 递进式摘要

**适用场景**：多问递进型赛题（问题1→2→3→4逐步深入）

**模板**：
```
针对问题1，运用[方法A]，求解[目标]，得到[结果]。
针对问题2，在问题1基础上进行[推广/改进]，引入[新变量/新方法]，得到[结果]。
针对问题3，首先[分析/判断]，再[建立模型]，利用[工具]求解，得到[结果]。
针对问题4，[将问题推广至更复杂情形]，采用[方法]，得到[结果]。
关键词：[3-6个关键词]
```

**要点**：
- 每个问题独立成段
- 方法→结果的因果链清晰
- 数值结果具体化（如"34条测线，总长度68海里"）

---

## Pattern 2: 框架式摘要

**适用场景**：整体性较强、方法贯穿多问的赛题

**模板**：
```
本文对[问题背景]进行研究。首先[建立基础模型]；针对[扩展条件]建立[扩展模型]；
在[更复杂情形]时，引入[新思想]，建立[高级模型]。给出不同情形下的[策略/方案]。

对于问题一：[具体方法与结果]
对于问题二：[具体方法与结果]
对于问题三：[具体方法与结果]

关键词：[3-6个关键词]
```

**要点**：
- 先总后分
- 强调模型间的递进关系
- 突出核心创新思想

---

## Pattern 3: 方法导向式摘要

**适用场景**：方法创新性强的论文

**模板**：
```
针对[问题背景]，本文基于[核心思想]建立了[模型体系]。
[方法1]用于[场景1]，[方法2]用于[场景2]，[方法3]用于[场景3]。
结果表明[核心结论]，对[应用领域]有[参考价值]。
关键词：[方法1] [方法2] [方法3] [应用场景]
```

---

## 关键规则

1. **数值必须具体**："覆盖率达99.7%"而非"覆盖率很高"
2. **方法必须命名**："基于动态规划的多阶段决策模型"而非"建立了一个模型"
3. **关键词要精准**：选择3-6个核心方法/概念词
4. **禁止空泛表述**：禁止"取得了良好效果"、"具有一定参考价值"等无信息量表述
5. **每问独立**：每个问题的结果必须可独立识别
