# /permission-check

## 用途
诊断权限提示来源：读取6个配置层，报告合并后的defaultMode和allow/deny列表

## 用法
```bash
/permission-check
```

## 检查内容
1. **配置层级** - 6个配置层
2. **默认模式** - defaultMode设置
3. **允许列表** - allow列表
4. **拒绝列表** - deny列表

## 配置层级
1. **系统级** - 系统默认配置
2. **用户级** - 用户配置
3. **项目级** - 项目配置
4. **目录级** - 目录配置
5. **文件级** - 文件配置
6. **会话级** - 会话配置

## 示例
```bash
/permission-check
```

## 输出
显示权限配置报告
