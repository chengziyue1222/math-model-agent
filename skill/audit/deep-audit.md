# /deep-audit

## 用途
全仓库深度一致性审计：4个并行代理（指南内容/代码质量/规则一致性/交叉一致性），修复所有问题并循环直到干净

## 用法
```bash
/deep-audit
```

## 审计维度
1. **指南内容** - 文档内容质量
2. **代码质量** - 代码规范和质量
3. **规则一致性** - 规则是否一致
4. **交叉一致性** - 跨文件一致性

## 审计流程
1. **并行审计** - 4个代理同时审计
2. **问题汇总** - 汇总所有问题
3. **修复问题** - 修复发现的问题
4. **循环检查** - 循环直到无问题

## 示例
```bash
/deep-audit
```

## 输出
生成审计报告并修复所有问题
