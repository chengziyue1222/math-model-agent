# /seven-pass-review

## 用途
七遍审稿法，系统性审查论文的各个方面

## 用法
```bash
/seven-pass-review manuscript.pdf
```

## 审稿七遍
1. **第一遍** - 标题和摘要
2. **第二遍** - 引言和结论
3. **第三遍** - 方法部分
4. **第四遍** - 结果部分
5. **第五遍** - 图表和表格
6. **第六遍** - 参考文献
7. **第七遍** - 整体连贯性

## 参数
- `filename`: 论文文件名

## 示例
```bash
/seven-pass-review manuscript.pdf
/seven-pass-review paper_draft.pdf
```

## 输出
生成详细的七遍审稿报告
