# /review-r

## 用途
R代码质量只读审查：代码质量、可复现性、领域正确性、tidyverse惯例

## 用法
```bash
/review-r scripts.R
/review-r all
```

## 审查维度
1. **代码质量** - 代码风格和规范
2. **可复现性** - 结果是否可复现
3. **领域正确性** - 统计方法是否正确
4. **tidyverse惯例** - 是否符合tidyverse风格

## 检查项目
1. **命名规范** - 变量和函数命名
2. **注释完整性** - 注释是否充分
3. **代码效率** - 代码性能
4. **错误处理** - 异常处理
5. **文档完整性** - 文档是否完整

## 参数
- `filename`: R脚本文件名或"all"

## 示例
```bash
/review-r analysis.R
/review-r all
```

## 输出
生成代码审查报告，不修改源文件
