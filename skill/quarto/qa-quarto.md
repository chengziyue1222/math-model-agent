# /qa-quarto

## 用途
Quarto vs Beamer对抗性质量对比（内容/视觉/符号一致性等6项硬门检查循环最多5轮）

## 用法
```bash
/qa-quarto Lecture
```

## 检查项目
1. **内容一致性** - 文本内容是否一致
2. **视觉一致性** - 布局和样式是否一致
3. **符号一致性** - 数学符号是否一致
4. **引用完整性** - 引用是否完整转换
5. **图表质量** - 图表是否清晰
6. **格式规范** - 格式是否符合要求

## 参数
- `filename`: 文件名

## 示例
```bash
/qa-quarto Lecture01
/qa-quarto Chapter03
```

## 迭代检查
支持最多5轮检查循环，每轮修复发现的问题后重新检查
