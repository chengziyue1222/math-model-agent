# /pedagogy-review

## 用途
整体教学法审查：叙事弧线、先修假设、示例、符号清晰度、整堂课节奏等13项检查

## 用法
```bash
/pedagogy-review Lecture1.tex
```

## 审查项目（13项）
1. **叙事弧线** - 课程逻辑流
2. **先修假设** - 学生背景知识假设
3. **示例质量** - 示例是否恰当
4. **符号清晰度** - 符号使用是否清晰
5. **节奏控制** - 课程节奏是否合理
6. **互动设计** - 互动环节设计
7. **练习设置** - 练习是否合适
8. **总结完整性** - 总结是否完整
9. **视觉辅助** - 视觉辅助效果
10. **难度梯度** - 难度是否循序渐进
11. **时间分配** - 时间分配是否合理
12. **学习目标** - 学习目标是否明确
13. **评估方式** - 评估方式是否有效

## 参数
- `filename`: 幻灯片文件名

## 示例
```bash
/pedagogy-review Lecture01.tex
/pedagogy-review Chapter03.tex
```

## 输出
生成13项教学法检查报告
