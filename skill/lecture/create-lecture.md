# /create-lecture

## 用途
从源论文创建完整Beamer讲座：多阶段协作流程

## 用法
```bash
/create-lecture 主题名称
```

## 创建流程
1. **论文分析** - 分析源论文内容
2. **结构建议** - 提供讲座结构建议
3. **逐批幻灯片** - 逐步创建幻灯片
4. **图形/代码** - 集成图形和代码
5. **最终润色** - 最终修改和完善

## 输出内容
1. **Beamer文件** - 完整的.tex文件
2. **图形文件** - 相关图形文件
3. **代码示例** - 代码示例
4. **讲义** - 配套讲义

## 参数
- `topic`: 讲座主题名称

## 示例
```bash
/create-lecture 机器学习基础
/create-lecture 计量经济学方法
```

## 输出
生成完整的Beamer讲座文件
