# /prompt

## 用途
将非正式输入重新格式化为结构化六段式并立即执行

## 用法
```bash
/prompt "你的想法"
```

## 六段式结构
1. **角色** - AI扮演的角色
2. **任务** - 具体任务描述
3. **上下文** - 相关背景信息
4. **约束** - 限制条件
5. **输出格式** - 期望的输出格式
6. **收尾** - 结束语

## 参数
- `input`: 非正式输入

## 示例
```bash
/prompt "帮我写一个数据分析报告"
/prompt "解释一下机器学习的概念"
```

## 输出
执行结构化的prompt并返回结果
