# /checkpoint

## 用途
保存结构化状态快照（活跃计划、决策、文件指针、开放问题、下一步行动）

## 用法
```bash
/checkpoint topic-slug
```

## 快照内容
1. **活跃计划** - 当前进行中的计划
2. **决策记录** - 已做出的决策
3. **文件指针** - 相关文件路径
4. **开放问题** - 待解决的问题
5. **下一步行动** - 后续行动

## 参数
- `slug`: 主题标识符

## 示例
```bash
/checkpoint 论文写作进度
/checkpoint 数据分析阶段
```

## 用途
- 会话中断后恢复
- 进度跟踪
- 知识管理
