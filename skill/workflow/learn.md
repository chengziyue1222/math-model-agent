# /learn

## 用途
将当前会话中发现的可用知识提取为持久skill文件

## 用法
```bash
/learn skill-name
```

## 功能说明
- 提取会话中的有价值知识
- 创建持久化的skill文件
- 便于后续复用
- 知识管理和积累

## 知识类型
1. **技术知识** - 编程和技术技巧
2. **方法知识** - 研究方法和流程
3. **工具知识** - 工具使用技巧
4. **经验知识** - 实践经验总结

## 参数
- `skill_name`: skill名称

## 示例
```bash
/learn latex-bibtex技巧
/learn 数据清洗流程
```

## 输出
生成skill文件，保存到skill目录
