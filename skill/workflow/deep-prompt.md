# /deep-prompt

## 用途
同prompt，但只生成prompt不执行，可保存为文件

## 用法
```bash
/deep-prompt "你的想法"
```

## 功能说明
- 生成结构化prompt
- 不立即执行
- 可保存为文件
- 便于复用和修改

## 六段式结构
1. **角色** - AI扮演的角色
2. **任务** - 具体任务描述
3. **上下文** - 相关背景信息
4. **约束** - 限制条件
5. **输出格式** - 期望的输出格式
6. **收尾** - 结束语

## 参数
- `input`: 非正式输入

## 示例
```bash
/deep-prompt "帮我设计一个研究方案"
/deep-prompt "创建一个数据分析流程"
```

## 输出
生成结构化prompt文件
