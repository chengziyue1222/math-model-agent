# Model Library

> 数学建模模型库。按问题类型组织，包含模型选择指南、实现要点、论文实例。

---

## 1. 几何分析类

### 1.1 三角函数+正弦定理

**适用场景**：涉及角度、距离、覆盖宽度的几何问题

**核心公式**：
- 正弦定理：a/sinA = b/sinB = c/sinC
- 余弦定理：c² = a² + b² - 2ab·cosC

**实现要点**：画出示意图→识别三角形→选择定理→建立方程求解

### 1.2 空间几何+向量分析

**适用场景**：三维空间中的角度、方向、投影问题

**核心方法**：方向向量表示方向、点积求角度、叉积求法向量、投影公式

## 2. 优化类

### 2.1 主要目标法

**适用场景**：多目标优化，目标有主次之分

**核心思想**：多目标 → 选最主要目标 → 其他目标转为约束 → 单目标优化

**实现要点**：识别所有目标→判断主次→次要转约束→求解→检查约束

### 2.2 贪心算法

**适用场景**：覆盖问题、调度问题、每次选局部最优

**实现要点**：定义贪心选择标准→证明贪心选择性质→迭代执行→验证结果

### 2.3 智能优化算法

| 算法 | 特点 | 适用场景 |
|------|------|---------|
| 遗传算法(GA) | 全局搜索强 | 组合优化、参数优化 |
| 粒子群(PSO) | 收敛快 | 连续优化 |
| 模拟退火(SA) | 跳出局部最优 | 组合优化 |
| 飞蛾火焰(MFO) | 改进空间大 | 连续优化 |
| 蚁群(ACO) | 离散优化 | 路径规划 |

## 3. 决策类

### 3.1 动态规划

**适用场景**：多阶段决策、离散时间、无后效性

**核心要素**：状态→决策→状态转移→收益→目标函数

**实现要点**：定义状态空间→定义决策空间→建立转移方程→建立目标函数→选择求解方向

### 3.2 马尔可夫决策过程(MDP)

**适用场景**：随机环境下的序贯决策

**与动态规划的区别**：

| 特征 | 动态规划 | MDP |
|------|---------|-----|
| 环境 | 确定型 | 随机型 |
| 转移 | 确定函数 | 概率分布 |
| 信息 | 完全信息 | 部分信息 |
| 求解 | 逆向递推 | 值迭代/策略迭代 |

### 3.3 博弈论

**适用场景**：多人竞争/合作决策

**核心概念**：玩家、策略、收益、纳什均衡、合作博弈 vs 非合作博弈

## 5. 微分方程与动态系统类

### 5.1 常微分方程数值解

| 方法 | 精度 | 适用场景 |
|------|------|---------|
| 欧拉法 | O(h) | 简单快速 |
| 改进欧拉法 | O(h²) | 中等精度 |
| 龙格库塔法(RK4) | O(h⁵) | 高精度 |
| ode45(Matlab内置) | 自适应步长 | 通用 |

**降阶技巧**：
```
原方程: m·x'' = f(t, x, x')
令 u = x, v = x'
则: u' = v, v' = f(t, u, v) / m
→ 两个一阶方程
```

### 5.2 遗传算法

**核心流程**：初始化种群→适应度评估→选择→交叉→变异→新种群→循环

## 6. 启发式搜索类

### 6.1 带阈值的启发式搜索

**适用场景**：组合优化、多步决策、状态空间搜索

**核心思想**：在IDA*算法启发下，每次决策前对未来若干层搜索，找局面最优的叶子节点

**与贪心/DFS的对比**：

| 方法 | 优点 | 缺点 |
|------|------|------|
| DFS | 全局搜索 | 效率极低 |
| 贪心 | 高效 | 短视，局部最优 |
| 启发式搜索 | 平衡效率与全局性 | 需设计估价函数 |

### 6.2 局部最优调整模型

**适用场景**：分布式系统、多Agent协调、位置调整

**核心思想**：每次只用局部信息调整，不依赖全局信息

## 7. 成分数据分析类

### 7.1 CLR变换（中心对数比变换）

**适用场景**：成分数据（比例数据，累加和为常数）

**变换公式**：`clr(x)_i = ln(x_i / g_m(x))`，其中 `g_m(x) = (∏ x_i)^(1/D)` 为几何平均

**关键性质**：等距变换（保持距离关系）、可逆、解释直观

### 7.2 成分数据分类

| 方法 | 特点 | 适用场景 |
|------|------|---------|
| SVM | 显式分类表达式 | 线性可分 |
| 决策树 | 可解释性强 | 多特征分类 |
| PLS-DA | 降维+分类 | 高维小样本 |
| K-means | 无监督聚类 | 亚类划分 |

**流程**：CLR变换→特征选择→SVM/决策树分类→敏感性分析
