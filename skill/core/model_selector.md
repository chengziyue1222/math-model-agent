# Model Selector Engine v2.0

>
> 输入：问题特征 → 输出：模型组合（主模型+辅助模型+验证模型）

## 决策规则格式

```
IF [问题特征]
THEN
  主模型: [核心求解方法]
  辅助模型: [数据预处理/参数估计/结果增强]
  验证模型: [交叉验证/敏感性分析/对比验证]
ELSE
  继续判断
```

## 一、规划类问题

### 1.1 线性规划 (LP)

**IF** 目标函数和约束条件均为线性函数，决策变量连续

**THEN**
- 主模型：线性规划（单纯形法/内点法）
- 辅助模型：灵敏度分析（参数变化对最优解的影响）
- 验证模型：图解法（2维时）/ 参数扫描

**求解工具**：`scipy.optimize.linprog` / `pulp`

**Python实现**：
```python
from scipy.optimize import linprog
# min c^T x, s.t. A_ub x <= b_ub, A_eq x = b_eq
result = linprog(c, A_ub=A, b_ub=b, bounds=bounds)
```

### 1.2 整数规划 (IP)

**IF** 部分或全部决策变量必须取整数值

**THEN**
- 主模型：整数规划（分支定界法/割平面法）
- 辅助模型：松弛线性规划（提供下界）
- 验证模型：枚举法（小规模时）

**特殊类型**：
- 0-1规划：决策变量只能取0或1（背包问题、指派问题）
- 混合整数规划：部分变量连续、部分整数

**求解工具**：`pulp` / `pyomo` + `glpk`/`cbc`

**Python实现**：
```python
import pulp
prob = pulp.LpProblem("IP", pulp.LpMinimize)
x = pulp.LpVariable.dicts("x", range(n), cat='Integer')
# 或 cat='Binary' 用于0-1规划
```

### 1.3 非线性规划 (NLP)

**IF** 目标函数或约束条件中至少有一个非线性函数

**THEN**
- 主模型：非线性规划（梯度下降/牛顿法/拟牛顿法）
- 辅助模型：多起点搜索（避免局部最优）
- 验证模型：网格搜索 / 随机搜索对比

**求解工具**：`scipy.optimize.minimize`（支持多种方法）

**Python实现**：
```python
from scipy.optimize import minimize
result = minimize(objective, x0, method='L-BFGS-B',
                  bounds=bounds, constraints=constraints)
```

### 1.4 多目标规划

**IF** 存在多个相互冲突的目标函数

**THEN**
- 主模型：根据目标关系选择
  - 有明显主次 → **ε约束法（主要目标法）**
  - 无明显主次 → **线性加权法** / **理想点法**
  - 有优先级 → **目标规划（序贯解法）**
- 辅助模型：目标函数无量纲化处理
- 验证模型：帕累托前沿分析 / 灵敏度分析

**ε约束法（主要目标法）**：
```
min f_k(x)
s.t. f_j(x) ≤ ε_j, j≠k
     x ∈ Ω
```
选择最重要目标为f_k，其余目标转为约束。

**目标规划**：
```
min Σ w_j * (d_j^+ + d_j^-)
s.t. f_j(x) + d_j^- - d_j^+ = b_j
     d_j^+, d_j^- ≥ 0
```

## 二、预测类问题

### 2.1 灰色预测 GM(1,1)

**IF** 数据量少（≥4个），数据呈指数增长趋势

**THEN**
- 主模型：GM(1,1)灰色预测
- 辅助模型：级比检验（数据预处理）
- 验证模型：残差检验 / 后验差检验

**建模步骤**：
1. 级比检验：计算级比λ(k)，检查是否在可容覆盖(e^(-2/(n+1)), e^(2/(n+1)))内
2. 累加生成：x^(1)(k) = Σx^(0)(i)
3. 建立微分方程：dx^(1)/dt + a*x^(1) = b
4. 参数估计：最小二乘法 [a,b]^T = (B^T B)^{-1} B^T Y
5. 求解预测值

**Python实现**：
```python
import numpy as np

def gm11_predict(x0, predict_n):
    """GM(1,1)灰色预测"""
    n = len(x0)
    # 级比检验
    lam = x0[:-1] / x0[1:]
    if not (np.exp(-2/(n+1)) < lam.min() and lam.max() < np.exp(2/(n+1))):
        print("Warning: 级比检验未通过，建议平移变换")
    # 累加生成
    x1 = np.cumsum(x0)
    # 构造数据矩阵B和Y
    B = np.column_stack([-0.5*(x1[:-1]+x1[1:]), np.ones(n-1)])
    Y = x1[1:]
    # 参数估计
    params = np.linalg.lstsq(B, Y, rcond=None)[0]
    a, b = params[0], params[1]
    # 预测
    result = np.zeros(n + predict_n)
    result[0] = x0[0]
    for k in range(1, n + predict_n):
        result[k] = (x0[0] - b/a) * np.exp(-a*k) + b/a
    # 还原
    predict = np.diff(result)
    predict = np.insert(predict, 0, x0[0])
    return predict[-predict_n:]
```

### 2.2 马尔可夫预测

**IF** 系统具有无后效性，状态转移概率已知或可估计

**THEN**
- 主模型：马尔可夫链（转移概率矩阵）
- 辅助模型：状态划分（等宽/等频/聚类）
- 验证模型：平稳分布分析 / 极限分布

**建模步骤**：
1. 状态划分：将连续数据离散化为有限状态
2. 估计转移概率矩阵：p_ij = n_ij / n_i
3. 预测：π(n+1) = π(n) * P

### 2.3 时间序列

**IF** 数据有时间顺序，需要短期预测

**THEN**
- 主模型：根据数据特征选择
  - 平稳序列 → ARIMA(p,d,q)
  - 有趋势 → 指数平滑（二次/三次）
  - 有季节性 → 季节性ARIMA / Holt-Winters
- 辅助模型：平稳性检验（ADF检验）、ACF/PACF定阶
- 验证模型：残差白噪声检验 / 滚动预测验证

### 2.4 回归预测

**IF** 需要建立因变量与自变量之间的关系

**THEN**
- 主模型：根据关系类型选择
  - 线性关系 → 多元线性回归
  - 非线性关系 → 多项式回归 / 核回归
  - 高维小样本 → 偏最小二乘回归(PLSR)
- 辅助模型：多重共线性检验（VIF）、异常值检测
- 验证模型：R²、调整R²、交叉验证

## 三、评价类问题

### 3.1 TOPSIS法

**IF** 需要对多个方案进行综合评价排序

**THEN**
- 主模型：TOPSIS法（逼近理想解排序法）
- 辅助模型：数据规范化（向量规范化/线性变换）、权重确定（熵权法/AHP）
- 验证模型：灵敏度分析（权重变化对排序的影响）

**算法步骤**：
1. 数据规范化（向量规范化）
2. 构造加权规范阵 C = w_j * b_ij
3. 确定正理想解C*和负理想解C^0
4. 计算各方案到正/负理想解的距离 d^+, d^-
5. 计算综合评价指数 C_j = d_j^- / (d_j^+ + d_j^-)
6. 按C_j排序

### 3.2 模糊综合评价

**IF** 评价指标具有模糊性，难以精确量化

**THEN**
- 主模型：模糊综合评价
- 辅助模型：隶属函数确定、权重向量确定
- 验证模型：多级模糊评价 / 灵敏度分析

**步骤**：
1. 确定因素集 U = {u_1, u_2, ..., u_n}
2. 确定评语集 V = {v_1, v_2, ..., v_m}
3. 确定权重向量 A = [a_1, a_2, ..., a_n]
4. 确定模糊综合判断矩阵 R
5. 综合评判 B = A · R

### 3.3 主成分分析 (PCA)

**IF** 变量多且存在相关性，需要降维

**THEN**
- 主模型：主成分分析
- 辅助模型：数据标准化、KMO检验
- 验证模型：碎石图、累计方差贡献率

### 3.4 数据包络分析 (DEA)

**IF** 评价多个决策单元的相对效率

**THEN**
- 主模型：DEA（C²R模型/BC²模型）
- 辅助模型：投入产出指标选择
- 验证模型：灵敏度分析 / Malmquist指数

### 3.5 灰色关联分析

**IF** 需要分析因素间的关联程度

**THEN**
- 主模型：灰色关联分析
- 辅助模型：数据无量纲化处理
- 验证模型：关联度排序的一致性检验

## 四、图与网络问题

### 4.1 最短路径

**IF** 需要求两点间的最短路径

**THEN**
- 主模型：根据图的特征选择
  - 非负权图 → Dijkstra算法 O(V²)
  - 所有点对 → Floyd算法 O(V³)
  - 含负权边 → Bellman-Ford算法 O(VE)
  - 有启发信息 → A*算法
- 辅助模型：图的邻接矩阵表示
- 验证模型：枚举法（小规模）/ 多算法交叉验证

### 4.2 旅行商问题 (TSP)

**IF** 需要找访问所有节点恰好一次的最短回路

**THEN**
- 主模型：精确解（分支定界）或近似解
  - 小规模 → 精确算法
  - 大规模 → 智能优化（遗传算法/模拟退火/蚁群）
- 辅助模型：改良圈算法（初始解生成）
- 验证模型：下界估计 / 多算法对比

### 4.3 最大流问题

**IF** 需要求网络中的最大流量

**THEN**
- 主模型：Ford-Fulkerson算法 / 最大流-最小割定理
- 辅助模型：残余网络构建
- 验证模型：最小割验证

### 4.4 关键路径法

**IF** 需要确定项目中的关键工序和工期

**THEN**
- 主模型：关键路径法(CPM) / 计划评审技术(PERT)
- 辅助模型：网络图绘制、时间参数计算
- 验证模型：甘特图 / 资源均衡分析

## 五、统计分析问题

### 5.1 参数估计与假设检验

**IF** 需要对总体参数进行推断

**THEN**
- 主模型：根据问题选择
  - 区间估计 → 置信区间（t分布/正态分布）
  - 假设检验 → t检验/卡方检验/F检验
- 辅助模型：正态性检验（Shapiro-Wilk）
- 验证模型：Bootstrap置信区间

### 5.2 聚类分析

**IF** 无标签数据，需要发现内在分组

**THEN**
- 主模型：根据数据特征选择
  - 样本量小 → 层次聚类（Q型聚类）
  - 样本量大 → K-means
  - 有噪声 → DBSCAN
- 辅助模型：距离度量选择（欧氏/马氏/切比雪夫）、数据标准化
- 验证模型：轮廓系数 / CH指数 / 聚类图

**层次聚类方法对比**：

| 方法 | 定义 | 特点 |
|------|------|------|
| 最短距离法 | D(G1,G2) = min d(xi,xj) | 易链式，适合紧凑类 |
| 最长距离法 | D(G1,G2) = max d(xi,xj) | 适合分离类 |
| 类平均法 | D(G1,G2) = 平均d(xi,xj) | 较均衡 |
| 离差平方和法(Ward) | D = D12 - D1 - D2 | 最常用，方差最小 |

### 5.3 回归分析

**IF** 需要建立变量间的定量关系

**THEN**
- 主模型：根据变量类型选择
  - 连续因变量 → 线性回归 / 多项式回归
  - 分类因变量 → Logistic回归
  - 高维数据 → 偏最小二乘回归(PLSR)
- 辅助模型：变量选择（逐步回归/AIC/BIC）、共线性诊断
- 验证模型：残差分析、交叉验证、R²

## 六、微分方程问题

### 6.1 常微分方程

**IF** 问题涉及连续时间动力学

**THEN**
- 主模型：根据方程类型选择
  - 可解析求解 → 分离变量法/积分因子法

- 辅助模型：高阶→一阶降阶处理
- 验证模型：步长收敛性检验 / 与解析解对比

**降阶技巧**：
```
x'' = f(t, x, x')  →  令 u=x, v=x'  →  u'=v, v'=f(t,u,v)
```

### 6.2 差分方程

**IF** 问题以离散时间形式出现

**THEN**
- 主模型：差分方程（递推关系）
- 辅助模型：初始条件确定
- 验证模型：与连续模型对比

**基本准则**：未来值 = 现在值 + 变化值

## 七、智能优化问题

### 7.1 模拟退火算法

**IF** 组合优化问题，需要跳出局部最优

**THEN**
- 主模型：模拟退火算法
- 辅助模型：邻域生成方法、冷却计划设计
- 验证模型：多次运行取最优 / 与精确解对比

**关键参数**：
- 初始温度T_0：足够大（接受概率>0.8）
- 降温策略：T_{k+1} = α * T_k，α∈[0.8, 0.99]
- 终止条件：温度足够低 或 连续多次无改善

### 7.2 遗传算法

**IF** 大规模优化、参数优化、组合优化

**THEN**
- 主模型：遗传算法
- 辅助模型：编码策略（十进制/二进制）、初始种群生成（改良圈算法）
- 验证模型：收敛性分析 / 与精确解对比

**关键参数**：
- 种群大小M：50-200
- 交叉概率pc：0.6-0.9
- 变异概率pm：0.01-0.1
- 进化代数G：100-1000

## 九、组合策略模板

对于复杂问题，通常需要多个模型组合：

### 递进式组合
```
基础模型 → 扩展模型 → 高级模型
例: 动态规划 → MDP → 博弈论
```

### 互补式组合
```
解析模型(建模) + 数值模型(求解) + 可视化(验证)
例: 几何分析 + 数值计算 + CAD验证
```

### 流水线式组合
```
数据预处理 → 核心建模 → 优化求解 → 结果验证
例: 标准化 + PCA + 回归 + 交叉验证
```

### 预处理+建模+验证组合
```
数据清洗 → 特征工程 → 模型训练 → 模型评估
例: 缺失值处理 + 特征选择 + SVM + 10-fold CV
```

**典型流程**: 预处理→去噪→增强→分割→形态学→特征提取→分类

**代码入口**: `from algorithms.image_processing import edge_detection, image_segmentation, feature_extraction`

## 十七、综合评价方法 (v6.0 新增)

**IF** 多指标综合评价 / 方案排序 / 效率评估 / 权重确定

**THEN**
- 主模型：根据评价类型选择
  - 方案排序 → `topsis` (逼近理想解)
  - 效率评估 → `dea` (数据包络分析)
  - 降维 → `pca` (主成分分析)
  - 客观权重 → `entropy_weight` (熵权法)
  - 主观权重 → `fahp` (模糊AHP)
  - 组合权重 → `combined_weight`
- 辅助模型：数据标准化、指标筛选
- 验证模型：灵敏度分析 / 权重扰动检验

**代码入口**: `from algorithms.evaluation import topsis, dea, pca, entropy_weight`

### 模型选择决策树（竞赛专用）

```
问题输入
├── 数据量少 (<20个)?
│   ├── 是 → 灰色预测 GM(1,1) / 灰色关联
│   └── 否 → 继续判断
├── 有优化目标?
│   ├── 线性约束 → 线性规划 / 整数规划
│   ├── 非线性 → 非线性规划 / 蒙特卡罗优化
│   ├── 多目标 → 加权法 / ε约束法
│   └── NP难 → 元启发式 (GA/SA/PSO)
├── 需要评价排序?
│   ├── 指标已知 → TOPSIS + 熵权法
│   ├── 指标模糊 → 模糊综合评价 + AHP
│   └── 效率评估 → DEA
├── 需要分类/聚类?
│   ├── 有标签 → SVM / 神经网络
│   └── 无标签 → K-means / 层次聚类
├── 需要预测?
│   ├── 时间序列 → ARIMA / 指数平滑
│   ├── 因果关系 → 回归分析
│   └── 非线性 → 神经网络
├── 涉及空间/扩散?
│   ├── 离散 → 元胞自动机
│   └── 连续 → 偏微分方程
└── 涉及图像?
    ├── 分析 → 边缘检测 + 特征提取
    └── 分类 → 图像处理 + 神经网络
```
