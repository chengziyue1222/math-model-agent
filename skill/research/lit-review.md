# /lit-review

## 用途
结构化文献综述：搜索→主题聚类→识别空白→生成BibTeX引用

## 用法
```bash
/lit-review "你的研究主题"
```

## 综述流程
1. **文献搜索** - 搜索相关文献
2. **主题聚类** - 按主题分类整理
3. **空白识别** - 识别研究空白
4. **引用生成** - 生成BibTeX引用

## 输出内容
1. **文献清单** - 相关文献列表
2. **主题分类** - 按主题聚类
3. **研究空白** - 识别的研究空白
4. **BibTeX文件** - 可直接使用的引用文件

## 参数
- `topic`: 研究主题

## 示例
```bash
/lit-review "机器学习在金融风控中的应用"
/lit-review "深度学习图像识别"
```

## 输出
生成结构化的文献综述报告和BibTeX文件
