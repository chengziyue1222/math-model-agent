# /devils-advocate

## 用途
5-7条教学性的对抗性挑战

## 用法
```bash
/devils-advocate Lecture1.tex
```

## 挑战维度
1. **顺序** - 逻辑顺序是否合理
2. **先修条件** - 假设是否合理
3. **空白** - 是否存在研究空白
4. **替代呈现** - 是否有更好呈现方式
5. **符号冲突** - 符号使用是否一致
6. **方法质疑** - 方法是否合理
7. **结论挑战** - 结论是否可靠

## 参数
- `filename`: 文件名

## 示例
```bash
/devils-advocate Lecture1.tex
/devils-advocate paper_draft.tex
```

## 输出
生成5-7条对抗性挑战问题，帮助改进研究
