# /research

## 用途
主题/现象/数据集生成3-5个结构化研究问题

## 用法
```bash
/research "你感兴趣的话题"
```

## 研究问题结构
每个研究问题包含：
1. **论文类型** - 理论/实证/综述
2. **假设** - 可检验的假设
3. **识别策略** - 因果识别方法
4. **数据需求** - 所需数据类型和来源

## 输出内容
1. **研究问题** - 3-5个结构化问题
2. **可行性分析** - 每个问题的可行性
3. **创新性评估** - 创新性评估
4. **资源需求** - 所需资源估算

## 参数
- `topic`: 研究话题

## 示例
```bash
/research "社交媒体对青少年心理健康的影响"
/research "气候变化与经济增长的关系"
```

## 输出
生成结构化的研究问题清单
