# /ideation

## 用途
互动式多轮访谈，将模糊的研究想法形式化为结构化规范

## 用法
```bash
/ideation "简要话题"
```

## 访谈内容
通过多轮访谈明确：
1. **研究问题(RQ)** - 明确的研究问题
2. **假设** - 可检验的假设
3. **识别策略** - 因果识别方法
4. **数据需求** - 所需数据
5. **实证策略** - 实证分析方法

## 访谈流程
1. **想法阐述** - 描述初步想法
2. **问题澄清** - 澄清模糊之处
3. **结构化** - 形式化为研究规范
4. **验证** - 验证可行性

## 参数
- `topic`: 研究话题

## 示例
```bash
/ideation "我想研究教育政策的影响"
/ideation "关于环境污染的经济学分析"
```

## 输出
生成结构化的研究规范文档
