# /interview-me

## 用途
互动式多轮访谈，将模糊的研究想法形式化为结构化规范

## 用法
```bash
/interview-me "简要话题"
```

## 访谈目标
通过访谈帮助用户：
1. **明确想法** - 将模糊想法具体化
2. **识别问题** - 发现潜在问题
3. **结构化** - 形成研究框架
4. **验证可行性** - 评估研究可行性

## 访谈问题类型
1. **背景问题** - 了解研究背景
2. **动机问题** - 明确研究动机
3. **方法问题** - 讨论研究方法
4. **预期结果** - 预期研究成果

## 参数
- `topic`: 研究话题

## 示例
```bash
/interview-me "我想做一个关于人工智能的研究"
/interview-me "关于教育公平的经济学分析"
```

## 输出
生成结构化的研究规范文档
